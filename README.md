## About The Project

The exponential growth of online misinformation has overwhelmed the capacity of human fact-checkers, creating an urgent need for automated verification systems. However, current AI approaches face a fundamental trade-off: traditional predictive models offer statistical stability but lack context, while generative models (like Gemini) provide flexible reasoning but are prone to hallucination.

Our solution is a novel, hybrid framework that synthesizes both worlds. By orchestrating a suite of specialized machine-learning classifiers and feeding their outputs into a Google Agent Development Kit (ADK) framework powered by **Gemini 3 Flash**, we generate a highly accurate, multi-dimensional credibility measure using a novel **Fractal Chain of Thought (FCoT)** prompting protocol.
